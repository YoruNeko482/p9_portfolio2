Portfolio YoruNeko

Hello World!

Este "trabajo" se realizó el primer día de clase cuando los conocimientos adquiridos eran mínimos. En este ejercicio descubrimos el uso y la aplicación del código HTML hNº (1-6).

Vínculos

Para nuestro segundo trabajo realizamos una práctica con la finalidad de aprender a usar los vínculos y conducir a la persona virtual de un sitio web a otro vinculado mediante el código a.

Usos Media

Usando el anterior ejercicio ampliamos este añadiendole diversos formatos de visualización como un vídeo mp4, un link a la plataforma de Youtube y un audio mp3. Usamos por primera vez los colores en los backgrounds y códigos para lograr que ciertos formatos se reprodujeran de forma automática en cuanto el visitante abre la página web. De la misma forma añadimos color a la tipografía y adornos como el subrayado.

Biblioteca

Empezamos con el uso del "display:flex;", usando esta característica del código HTML para realizar una biblioteca con diversos apartados donde podemos observar diversos géneros de manga (estilo de dibujo oriental). Hicimos uso de links para crear un camino hacia diferentes redes sociales, vinculamos los diversos apartados de este proyecto en un sencillo índice al inicio de la página y usamos el "flex-wrap" para mejorar la minimación y maximación de nuestra página.

Grid Face

Esta sencilla actividad se realizó con el objetiv de aprender a ejecutar de manera eficaz el aspecto de grid, característica que nos permite establecer una cuadrícula en la cual situar los elementos que requiramos. Además del uso del grid también se usó el aspecto del css @media con el cual al establecer diferentes medidas lograbamos ejecutar diversos cambios cuando el tamaño del sitio web se reducía o se agrandaba hasta el punto establecido.

Neoplasticismo

En esta actividad hemos recreado un cuadro neoplasticista del artista Mondrian usando la herramienta del grid. Dentro de esta vinculamos un pequeño índice a los bocetos de esta actividad.

1r Portfolio

Este fue el 1r portfolio que realizamos. Usando el grid para la estructuración de las diversas actividades, el background-image para el uso de fotografias cómo fondo, el :hoover y @media para los diferentes formatos de pantalla, realizamos este proyecto para establecer un sitio donde poder acceder a todos nuestros trabajs de forma intuitiva e informativa al mismo tiempo que creativa i personificada.

Animación y transición

Una de las últimas prácticas que hemos realizado es la adición de animaciones a nuestras páginas web, el uso de la herramienta transform y @keyframe nos permiten realizar gran diversidad de acciones dinámicas. También practicamos con las diferentes modificaciones para :hoover.

Portfolio

Finalmente este es el trabajo actual, un portfolio para el cual buscamos referencias de otros trabajos similares ya creados y posteriormente personalizamos esa plantilla para realizar nuestro propio proyecto.
